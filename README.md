# 🛒 SuperPlus

Tu lista familiar inteligente. App colaborativa para gestionar el supermercado en pareja.

## Features
- 📋 Lista compartida en tiempo real
- 📦 Catálogo de productos configurables
- 🔮 Predicciones de compra inteligentes
- 🛒 Modo compra (vista simplificada para la tienda)
- 🧾 Escaneo de tickets con IA
- 📊 Historial completo con precios y fechas

## Stack
- React + Vite
- Supabase (PostgreSQL + Auth)
- Vercel (hosting)

## Desarrollo local
```bash
npm install
npm run dev
```
